# Task: primality test

In `src/sol.jl`, implement `is_prime(n)` that returns `1` (as an `Int`) if `n` is
prime and `0` otherwise. Numbers less than 2 are not prime.

Signature: `function is_prime(n)` (where `n` is an `Int`).

Examples: `is_prime(2)` = 1, `is_prime(7)` = 1, `is_prime(9)` = 0,
`is_prime(1)` = 0, `is_prime(13)` = 1.
