# Task: count even numbers

In `src/sol.jl`, implement `count_even(xs)` that returns, as an `Int`, how many
elements of a vector of integers are even. An empty vector has `0` evens.

Signature: `function count_even(xs)` (where `xs` is a `Vector{Int}`).

Examples: `count_even([1, 2, 3, 4, 6])` = 3, `count_even([1, 3, 5])` = 0,
`count_even(Int[])` = 0.
