# Task: primality test

In a module named `Sol` (file `src/Sol.obn`), implement

```
PROCEDURE IsPrime*(n: INTEGER): INTEGER
```

returning `1` if `n` is prime and `0` otherwise. (Returns an INTEGER because
every procedure in this suite returns INTEGER.) Recall `1` is not prime and the
smallest prime is `2`.

Examples: `IsPrime(2)` = 1, `IsPrime(7)` = 1, `IsPrime(9)` = 0,
`IsPrime(1)` = 0, `IsPrime(13)` = 1.
