# Task: largest element of a list

Implement a function that returns the maximum value of a non-empty list of
integers.

In `src/sol.gleam` (module `sol`), implement:

```gleam
pub fn largest(xs: List(Int)) -> Int
```

Examples: `largest([3, 1, 2])` = 3, `largest([5])` = 5, `largest([4, 4, 4])` = 4.
