# Task: greatest common divisor

Implement a function that returns the greatest common divisor of two
non-negative integers `a` and `b` (`gcd(n, 0) = n`).

In `src/sol.gleam` (module `sol`), implement:

```gleam
pub fn gcd_of(a: Int, b: Int) -> Int
```

Examples: `gcd_of(12, 18)` = 6, `gcd_of(7, 0)` = 7, `gcd_of(0, 5)` = 5,
`gcd_of(17, 5)` = 1.
